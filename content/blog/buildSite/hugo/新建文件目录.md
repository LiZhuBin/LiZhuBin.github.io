# 建立文件目录发
每个目录中都应该有_index.md文件
用于显示该目录下文件的说明
## 建立普通文件
~~~
hugo new (地址/文件名).md
~~~
## 建立目录文件
~~~
hugo new --kind chapter (地址/文件名).md
~~~
**会生成相应的默认数据**
~~~
title = "建立文件目录"
date = 2019-04-28T13:22:24+08:00
weight = 2
chapter = true
~~~
**其中weight 表示目录的排列顺序 （越小越排前）**
