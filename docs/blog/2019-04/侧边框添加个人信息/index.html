侧边框添加个人信息

操作一

将要隐藏的，不显示在侧边框的文件放在content根目录下

操作二

在 config.toml 中添加个人信息

name : 显示文字

identifier: 唯一标志 url ： 链接（本地或网络）

[[menu.shortcuts]]
name = "<i class='fab fa-github'></i> Github repo"
identifier = "ds"
url = "https://github.com/matcornic/hugo-theme-learn"
